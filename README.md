## Анубис
Ану́бис (др.-греч. Ἄνουβις, егип. jnpw Инпу) — древнеегипетский бог погребальных ритуалов и мумификации (бальзамирования), «страж весов» на суде Осириса в царстве мёртвых, знаток целебных трав.
